Tag Name Description

<a>

 Anchor

Creates a hyperlink.

<p>

Paragraph

Renders text.

<header>

Header

A semantic division that contains a header to the web page.

<nav>

Navigation

A semantic division that contains the navigation hyperlinks.

<aside>

Aside

A semantic division that contains useful (but not main) information.

<section>

Section

A semantic division that contains a section of data inside the main part of the page.

<footer>

Footer

A semantic division that contains footer information, about the website owner/creator.

<address>

Address

A semantic division that contains contact information.
